Bot API Reference
How the Discord bot updates statuses in Firestore.
How status is determined
The bot scans Discord channels every 5 minutes. 🟢 channel -> online (executor is up) 🔴 channel -> offline (executor is down) 🟡 channel -> verifying (awaiting human update) After a Roblox update, all channels are auto-set to 🔴. After 3 hours, 🔴 channels auto-transition to 🟡. A human with Manage Channels can run /force green to confirm recovery.

Bot slash commands
/force <red|yellow|green> — force all channels to one color /executor <name> <status> — override one executor manually /scan — trigger an immediate scan now /incident — create or update an incident /maintenance — schedule a maintenance window /end <Incident|Maintenance> <id> — resolve/complete with captcha confirm /adminrole <role> — set the admin role (server owner only)
